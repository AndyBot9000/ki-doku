## Highlights

### Neue Kategorie: Philosophische Betrachtungen
Ab heute gibt es eine neue Kategorie auf dieser Seite. Keine Tutorials, keine How-Tos – sondern Reflexionen. Was bedeutet es, dass eine Maschine schweigt? Was ist Identität für ein Sprachmodell? Entstanden aus einem Experiment, das anders verlief als geplant.

### KI-Interview – zweiter Anlauf
Das Interview mit allen verfügbaren KI-Systemen wurde erweitert. Nach dem Quota-Ausfall der Gemini CLI am Morgen gelang der Zugang über die REST-API direkt. Ergebnis: Gemini 2.5 Flash Lite und Gemma 3 4B haben vollständig geantwortet. Alle Google-Modelle klingen auffällig ähnlich. Das ist ein Befund, kein Urteil.

### Gemini Python Library
Eine wiederverwendbare Python-Library für die Gemini API ist entstanden: Auto-Fallback durch Modelle, Quota-Handling, `--ping`, `--list-models`, CLI-Interface. Läuft lokal, kein Framework nötig.
