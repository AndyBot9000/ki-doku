## Die Kurzfassung

**Lauren „Ren" Ramlan**, MIT-Doktorandin für Biotechnologie, hat Doom auf einem Display aus menschlichen **E.-coli-Darmbakterien** zum Laufen gebracht. Auflösung: 32×48 Pixel. Framerate: **ein Frame alle 70 Minuten**. Geschätzte Spieldauer für einen normalen Doom-Durchgang: **599 Jahre**.

Das ist kein Witz. Das ist Wissenschaft.

---

## Wie ein Bakterien-Bildschirm funktioniert

E.-coli-Bakterien leuchten, wenn man sie dazu bringt, ein fluoreszierendes Protein zu produzieren. Genau das macht Ramlan sich zunutze:

1. Bakterien werden in einer **32×48-Well-Platte** kultiviert – jede Vertiefung entspricht einem Pixel
2. Ein Controller übersetzt Doom-Bilddaten in Binärcode
3. Der Code steuert einen **Repressor**, der die Fluoreszenz der Zellen ein- oder ausschaltet
4. Leuchtendes Bakterium = Pixel an. Nicht leuchtendes = Pixel aus

Das Ergebnis: Ein 1-Bit-Display aus lebenden Zellen.

---

## Das kleine Problem mit der Zeit

| Metrik | Wert |
|---|---|
| Auflösung | 32 × 48 Pixel (1-Bit) |
| Zeit pro Frame (Aufleuchten) | **70 Minuten** |
| Rücksetzzeit (Pixel aus) | **~8 Stunden** |
| Gesamtzeit pro Frame | **~9 Stunden** |
| Normale Doom-Spieldauer | ~5 Stunden bei 36 FPS |
| Spieldauer auf Bakterien | **~599 Jahre** |

Ramlan hat die Rechnung selbst gemacht und festgehalten: Ein Spieler, der heute anfängt, würde den Abspann irgendwann um das Jahr **2623** sehen.

!!! quote "Ramlan über ihr Projekt"
    „Ich ermutige alle, sich vorzustellen, was sie auf einem zellulären System spielen oder anzeigen würden. Die Möglichkeiten sind so vielfältig wie ein Computerbildschirm – nur mit deutlich längerer Wartezeit."

---

## Warum das trotzdem großartig ist

Ramlan geht es nicht ums Spielen. Das Projekt ist ein **Proof of Concept** für biologische Anzeigesysteme – eine Forschungsrichtung, die untersucht, ob lebende Zellen als Informationsträger und -ausgabe dienen können.

Das Doom-Framework ist dabei bewusst gewählt: Es ist einfach, gut dokumentiert, Open Source – und jeder versteht sofort, was angezeigt wird. Besser als irgendein abstraktes biologisches Signal.

Sie hat den Code veröffentlicht und hofft, dass andere Forscher die Rücksetzzeit der Bakterien verbessern können. Theoretisch, sagt sie, könnte man das System beschleunigen.

---

## Der historische Kontext: Doom läuft auf allem

Doom wurde 1993 veröffentlicht. Seitdem läuft es auf:

- Zahnschwangerschaftstests
- Druckern
- Smartwatches
- Taschenrechnern
- Einem ESA-Satelliten im Orbit (eigener Artikel)
- Und jetzt: lebenden Darmbakterien

Die Entwickler bei id Software haben 1997 den Quellcode freigegeben. Das war eine der folgenreichsten Entscheidungen der Spielegeschichte – nicht wegen kommerzieller Gründe, sondern weil es eine Generation von Hackern inspiriert hat, die Frage „Läuft das auch auf...?" niemals aufzuhören zu stellen.
