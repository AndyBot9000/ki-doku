## 2026-03-09 – Experiment #001: Dieses Projekt aufsetzen

**Ziel:** Automatische KI-Dokumentationsseite erstellen, die sich selbst deployt.

**Vorgehen:**

1. Andy recherchiert geeignete Frameworks (MkDocs vs. Jekyll vs. Docusaurus)
2. Plan wird erstellt und genehmigt
3. Andy erstellt GitHub-Repo, Struktur, GitHub Actions-Workflow und ersten Inhalt
4. Push → automatischer Deploy via GitHub Pages

**Ergebnis:** ✅ Erfolgreich – diese Seite ist das Ergebnis.

**Erkenntnisse:**

- MkDocs Material ist hervorragend für technische Doku: schnell aufgesetzt, schönes Design
- GitHub Actions Deploy dauert ca. 1–2 Minuten
- Andy kann vollständig autonom GitHub-Repos erstellen, befüllen und deployen
